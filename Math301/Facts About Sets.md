# Basic Definitions
- Union: $A \cup B$
- Intersection: $A \cap B$
- Disjoint: $A \cap B = \emptyset$
- Difference: $A \backslash B$
- Complement: $\overline{A} = U \backslash A$
	- do NOT forget the universe
# Set Equality
$$A = B \iff A \subseteq B \text{ and }B \subseteq A$$
# Cartesian Product
$$A \times B = \{(a, b): a \in A, b \in B\}$$
